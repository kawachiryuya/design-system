# Atomsコンポーネント一覧

## 定義
**これ以上分解できない最小のUI要素**

---

## 優先度順

### 🔴 Phase B-1: 最優先

1. **Button** - クリック可能な要素
2. **Input** - テキスト入力
3. **Label** - フォームラベル  
4. **Icon** - アイコン表示
5. **Typography** - テキストスタイル

### 🟡 Phase B-2: 高優先度

- Checkbox / Radio / Select / Textarea / Switch / Badge / Spinner

### 🟢 Phase B-3: 中優先度

- Link / Avatar / Divider / Image / ProgressBar / Skeleton

---

## 推奨: Button から始める

最も使用頻度が高く、トークン・状態管理・アクセシビリティの基礎を学べる。
