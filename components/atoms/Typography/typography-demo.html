as prop — 意味論と視覚の分離

視覚スタイル（variant）と HTML 要素（as）を独立して指定できます。見出し階層を崩さずにスタイルを適用する場合に有効です。

variant="h1" as="h2" H1スタイル / H2タグ

variant="h2" as="h3" H2スタイル / H3タグ

variant="h3" as="p" H3スタイル / pタグ（見出しではない）

Truncate（テキスト省略）

truncate なし

非常に長いタイトルが入ることがあり、この場合はテキストが折り返されて複数行になります。

truncate あり

非常に長いタイトルが入ることがあり、この場合はテキストが省略されて一行に収まります。

実践例: 記事レイアウト

デザイン原則

タイポグラフィで情報の階層を伝える

文字は単なる情報の容れ物ではなく、階層・重要度・感情を伝えるデザイン要素です。

2026年2月21日 · 5分で読めます

タイポグラフィは、視覚デザインの中で最も影響力の大きい要素の一つです。適切な階層を設定することで、ユーザーは素早く情報をスキャンし、必要な内容を見つけられます。

見出しのルール

見出しは論理的な順序で使用します。h1 → h2 → h3 の順で、スキップは禁止です。

※ WCAG 2.1 AA に基づく実装が必要です。
