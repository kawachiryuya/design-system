## 原則の定義

**グレースケールを定義し、テキスト、背景、境界線に一貫して使用する**

## 用途別の推奨

### テキスト

| 用途 | 色 | コントラスト比 |
|------|-----|--------------|
| 主要テキスト | neutral-900 | 16:1 |
| 副次的テキスト | neutral-700 | 8:1 |
| 補足テキスト | neutral-600 | 5:1 |
| Disabled | neutral-400 | 3:1 |

### 背景

| 用途 | 色 |
|------|-----|
| ページ背景 | white / neutral-50 |
| カード背景 | white |
| Hover背景 | neutral-50 / neutral-100 |
| Active背景 | neutral-100 / neutral-200 |

### 境界線

| 用途 | 色 |
|------|-----|
| 標準 | neutral-200 |
| Focus | neutral-400 / brand-600 |
| Error | error-600 |

## ダークモード

ライト/ダークで値を反転:
- neutral-900 ↔ neutral-50
- neutral-700 ↔ neutral-300
