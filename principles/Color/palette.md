# カラーパレット

## メタ情報
- カテゴリ: カラー原則
- 適用範囲: 全階層
- ステータス: Approved
- 最終更新: 2024-01-15

## 原則の定義

**50〜900の段階的なスケールでカラーパレットを定義し、明度に応じた適切な色を選択できるようにする**

## スケールの構造

```
50  - 最も明るい（背景、ハイライト）
100
200
300
400
500 - ベースカラー
600
700
800
900 - 最も暗い（テキスト、強調）
```

## 使い分けの目安

| スケール | 用途 | 例 |
|---------|------|-----|
| 50-100 | 背景、subtle | ホバー背景、通知背景 |
| 200-300 | Border、disabled | ボーダー、無効状態 |
| 400-500 | UI要素 | アイコン、グラフ |
| 600-700 | Primary | ボタン、リンク |
| 800-900 | テキスト、強調 | 見出し、強調テキスト |

## コントラスト考慮

- 50-300: 白背景でテキストに不適（コントラスト不足）
- 600-900: 白背景でテキストに適切
- 50-200: 黒テキストに適切な背景

→ 詳細は [Color Contrast](../foundation/accessibility/color-contrast.md)
