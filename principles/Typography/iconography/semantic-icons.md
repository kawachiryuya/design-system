# セマンティックアイコン

## メタ情報
- カテゴリ: コンテンツ原則 > Iconography

- ステータス: Approved
- 最終更新: 2024-02-17

## 原則の定義

**意味を持つアイコンはシステム全体で統一し、色と組み合わせて確実に伝える**

セマンティックアイコンとは、Success・Error・Warning・Info等、特定の意味を持つアイコンです。色と組み合わせることで、テキストを読まなくても状態が伝わります。

## 標準セマンティックアイコン

### 状態アイコン

| 意味 | アイコン | 色 | 用途 |
|------|---------|-----|------|
| Success（成功） | ✓ チェックサークル | success-600（緑） | 完了・正常・承認 |
| Error（エラー） | ✕ バツサークル | error-600（赤） | 失敗・エラー・拒否 |
| Warning（警告） | ⚠ 三角 | warning-600（黄橙） | 注意・リスク |
| Info（情報） | ℹ インフォ | info-600（青） | ヒント・補足情報 |

### UI操作アイコン

| 意味 | アイコン | 用途 |
|------|---------|------|
| 閉じる | × | モーダル・タグ・アラートの閉じる |
| 戻る | ← | ナビゲーション・前のページ |
| 展開 | ∨ | アコーディオン・ドロップダウンの開閉 |
| 設定 | ⚙ | 設定・オプション |
| 検索 | 🔍 | 検索フィールド |
| 編集 | ✏ | 編集モード |
| 削除 | 🗑 | 削除アクション |
| 追加 | + | 追加・新規作成 |

## 使用ルール

### 色と必ずセットで使う

状態アイコンは、色だけでなくアイコンも必ず一緒に使う。逆もしかり。

```
❌ 悪い例: 色だけ（色覚異常のユーザーに伝わらない）
  「● 保存しました」（緑のドットのみ）

❌ 悪い例: アイコンだけ（スクリーンリーダーに伝わらない）
  「✓ 保存しました」（アイコンにaria-hiddenなし）

✅ 良い例: アイコン + 色 + テキスト
  [✓] 保存しました（アイコン緑 + テキスト）
```

→ [Semantic Colors](../../color/semantic-colors.md) 参照

### システム内で意味を固定する

```
✅ 一貫した使い方:
  ✓（チェック）= 常にSuccessを意味する

❌ 矛盾する使い方:
  ✓ = ページAでは「完了」、ページBでは「選択中」
  → 同じアイコンが異なる意味 → 混乱
```

### 削除・破壊的アクションのアイコン

削除アイコン（🗑）には特別な注意が必要。

```
✅ 削除ボタン: エラー色（red）+ 🗑 アイコン
  「[🗑] 削除」→ 危険なアクションであることが伝わる

確認ダイアログも必ず表示する。
```

→ [Placement](../../interaction/button/placement.md) 参照

## アクセシビリティ

### 状態アイコンの読み上げ

```html
<!-- アラートコンポーネントの例 -->
<div role="alert" class="alert-error">
  <!-- アイコンは装飾（テキストで意味を伝えるため aria-hidden） -->
  <ErrorIcon aria-hidden="true" />
  <span>保存に失敗しました。もう一度お試しください。</span>
</div>
```

### アイコン単独の場合（ラベル必須）

```html
<!-- 閉じるボタン -->
<button aria-label="ダイアログを閉じる">
  <CloseIcon aria-hidden="true" />
</button>

<!-- 削除ボタン -->
<button aria-label="このアイテムを削除">
  <TrashIcon aria-hidden="true" />
</button>
```

→ [Screen Readers](../../foundation/accessibility/screen-readers.md) 参照

## チェックリスト

- [ ] 状態アイコンが色と組み合わさっている
- [ ] 同じアイコンが異なる意味で使われていない
- [ ] 削除アイコンはエラー色と組み合わせている
- [ ] アイコン単独の場合にaria-labelを付与
- [ ] テキストと組み合わせる場合はaria-hiddenを付与
