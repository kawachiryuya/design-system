# 代替テキスト（Alt Text）

## メタ情報
- カテゴリ: コンテンツ原則 > Imagery

- ステータス: Approved

## 原則の定義

**画像が伝える情報をテキストで正確に伝え、スクリーンリーダーユーザーや画像が表示されない環境でも同等の体験を提供する**

## 2種類の画像と対応

すべての `<img>` には `alt` 属性が必要。ただし内容は画像の種類によって異なる。

### 1. 情報を伝える画像（Informative Image）

画像自体がコンテンツの一部。`alt` に意味のある説明を書く。

```html
<!-- 商品画像 -->
<img src="shoe.jpg" alt="ナイキ エアマックス 90 ホワイト/ブラック" />

<!-- グラフ・チャート -->
<img src="chart.jpg" alt="2024年Q1〜Q4の売上推移。Q3に最高値（1.2億円）を記録" />

<!-- 人物写真（コンテキストに応じて） -->
<img src="team.jpg" alt="オフィスでミーティングをする4人のチームメンバー" />
```

### 2. 装飾的な画像（Decorative Image）

背景・雰囲気づくりのみで情報を持たない画像。`alt=""` を設定してスクリーンリーダーにスキップさせる。

```html
<!-- 装飾的な背景画像 -->
<img src="bg-pattern.jpg" alt="" role="presentation" />

<!-- テキストの隣に置いた装飾アイコン（テキストが意味を伝える） -->
<img src="star-icon.svg" alt="" /> おすすめ
```

**判断基準**:
```
「この画像が消えたら、意味が失われるか？」
  Yes → 情報を伝える画像 → altに説明を書く
  No  → 装飾的な画像 → alt=""
```

## altテキストの書き方

### 内容を説明する（どう見えるかではなく、何を伝えるか）

```
❌ 悪い例（見た目の説明）:
「青い背景に白い文字が書かれた画像」

✅ 良い例（情報の説明）:
「送料無料キャンペーン実施中」
```

### 簡潔に（長くなる場合は構造化する）

```
✅ 短い画像（〜100文字）:
<img src="logo.png" alt="株式会社サンプル" />

✅ 複雑な画像（図解・チャート）:
<figure>
  <img src="flowchart.png" alt="ユーザー登録フローの概要" />
  <figcaption>
    詳細: ①メール入力 → ②確認メール送信 → ③リンクをクリック → ④プロフィール設定 → ⑤完了
  </figcaption>
</figure>
```

### 「画像の」「写真の」は不要

スクリーンリーダーは `<img>` を「画像」と読み上げるので重複する。

```
❌ 「〇〇の画像」「〇〇の写真」
✅ 「〇〇」だけでよい
```

### テキストが含まれる画像

画像内のテキストはaltにそのまま書く。

```html
<!-- ボタンのスクリーンショット -->
<img src="button-example.png" alt="「今すぐ始める」ボタン" />

<!-- バナー画像 -->
<img src="banner.jpg" alt="春の新生活セール 最大50%OFF 4月30日まで" />
```

## コンテキスト別のガイドライン

### アバター・プロフィール画像

```html
<!-- 本人のプロフィールページ -->
<img src="avatar.jpg" alt="山田太郎のプロフィール写真" />

<!-- コメント欄など、名前がすでに表示されている場合 -->
<img src="avatar.jpg" alt="" />  ← 名前が別途表示されているので装飾扱い
```

### リンクの中にある画像

リンクのアクションを説明する。

```html
<!-- テキストなしのアイコンリンク -->
<a href="/home">
  <img src="home-icon.svg" alt="ホームへ戻る" />
</a>

<!-- テキスト付きリンク内の装飾アイコン -->
<a href="/home">
  <img src="home-icon.svg" alt="" />
  ホーム
</a>
```

### 商品・サービス画像

```html
<!-- 主要な商品画像（情報量が多い） -->
<img src="product.jpg" alt="ステンレス製タンブラー 500ml シルバー。二重構造で保温・保冷対応" />

<!-- サブ画像（メイン画像の補足） -->
<img src="product-side.jpg" alt="タンブラー側面。スリムなフォルムで持ちやすい形状" />
```

## チェックリスト

- [ ] すべての `<img>` に `alt` 属性がある
- [ ] 装飾的な画像は `alt=""`
- [ ] 情報を伝える画像に意味のある説明がある
- [ ] 「〜の画像」「〜の写真」という冗長表現がない
- [ ] 画像内のテキストがaltに含まれている
- [ ] リンク内の画像にリンクの目的が書かれている
