# アスペクト比

## メタ情報
- カテゴリ: コンテンツ原則 > Imagery

- ステータス: Approved

## 原則の定義

**画像のアスペクト比を統一し、レイアウトの一貫性とガタつきのない読み込みを実現する**

## なぜアスペクト比を統一するか

```
❌ バラバラな比率:
┌──────────┐ ┌────┐ ┌──────┐
│  画像A   │ │B   │ │ 画像C │
│ 16:9     │ │3:4 │ │ 1:1  │
└──────────┘ └────┘ └──────┘
→ グリッドが崩れ、視線が定まらない

✅ 統一した比率:
┌──────────┐ ┌──────────┐ ┌──────────┐
│  画像A   │ │  画像B   │ │  画像C   │
│ 16:9     │ │ 16:9     │ │ 16:9     │
└──────────┘ └──────────┘ └──────────┘
→ グリッドが整い、一貫性がある
```

## 標準アスペクト比の定義

| 比率 | 用途 |
|------|------|
| **16:9** | ヒーロー画像・バナー・動画サムネイル |
| **4:3** | 汎用コンテンツ画像・ブログサムネイル |
| **1:1** | プロフィールアイコン・SNS投稿・商品画像 |
| **3:4** | 人物ポートレート・縦長コンテンツ |
| **2:1** | ワイドバナー・ランディングページ |

## 実装

### CSS aspect-ratio

```css
/* ✅ 推奨: CSS aspect-ratioで制御 */
.card-image {
  aspect-ratio: 16 / 9;
  width: 100%;
  object-fit: cover;     /* 比率を保ちながら領域を埋める */
  object-position: center; /* 中央を基準にトリミング */
}

/* 縦横中央寄せにしたい場合 */
.profile-image {
  aspect-ratio: 1 / 1;
  object-fit: cover;
  object-position: top center; /* 顔が上にある場合 */
  border-radius: 50%;
}
```

### レイアウトのガタつき防止（CLS対策）

```html
<!-- width / height を指定してレイアウトシフトを防ぐ -->
<img
  src="hero.jpg"
  width="1600"
  height="900"
  alt="..."
  style="width: 100%; height: auto;"
/>
```

`width` と `height` 属性を設定することでブラウザが事前にスペースを確保し、画像読み込み後のガタつきを防ぐ。

## object-fit の使い分け

```css
object-fit: cover;    /* 比率を保ち、領域いっぱいに（はみ出る部分はトリミング） */
object-fit: contain;  /* 比率を保ち、領域に収まるよう縮小（余白が生まれる） */
object-fit: fill;     /* 比率を無視して領域を埋める（歪む） ← 原則使わない */
```

```
cover（推奨）:           contain:              fill（非推奨）:
┌────────────┐          ┌────────────┐        ┌────────────┐
│▓▓▓▓▓▓▓▓▓▓▓│          │░░░░░░░░░░░│        │▓▓▓▓▓▓▓▓▓▓▓│
│▓▓▓▓▓▓▓▓▓▓▓│          │░┌────────┐│        │▓▓▓▓▓▓▓▓▓▓▓│
│▓▓▓[画像]▓▓│          │░│  画像  │░│        │▓▓[画像(歪)]│
│▓▓▓▓▓▓▓▓▓▓▓│          │░└────────┘│        │▓▓▓▓▓▓▓▓▓▓▓│
└────────────┘          └────────────┘        └────────────┘
```

## チェックリスト

- [ ] 同じコンテキストの画像が統一したアスペクト比
- [ ] `object-fit: cover` で統一（fill は使わない）
- [ ] `width` / `height` 属性を設定（CLS対策）
- [ ] `aspect-ratio` CSS プロパティで制御
